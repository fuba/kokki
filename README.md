# 国旗生成ゲーム

ランダムな国旗を生成し、損壊するブラウザゲームです。

**https://fuba.github.io/kokki/**

## 遊び方

1. 「生成」ボタンを押すとルーレットが回り、ランダムな国旗が生成されます
2. 生成された国旗は爆発または燃焼エフェクトで損壊されます
3. 結果ダイアログで損壊の様子をGIFアニメーションとして確認できます
4. GIF画像を保存したり、X（Twitter）に共有できます

## 特徴

### 国旗生成

4種類の図形（円・星・三日月・十字）と5色の背景色・図形色の組み合わせからランダムに生成します。

### 実在国旗の検出

生成された国旗が実在の国旗に近い場合、特別な演出が発生します。

| 国旗 | 演出 |
|------|------|
| 日本 | 君が代を演奏後、損壊。国旗損壊罪法案についての警告を表示 |
| スイス、バングラデシュ、パラオ、ベトナム、ソマリア、トルコ、パキスタン、チュニジア | 各国の国歌を演奏。国旗は損壊されず波打つGIFを表示。刑法第92条の警告を表示 |

### 損壊エフェクト

- **爆発**: 3,000個のパーティクルが放射状に飛散
- **燃焼**: 下部から上部へ炎が燃え広がり、国旗が焦げて崩壊

### WebGL描画

- 3Dの旗が風になびくアニメーション
- 時刻に連動した空の色変化（朝焼け・昼・夕焼け・夜）
- 金属製のポールとロープ
- ドラッグでカメラアングルを変更可能

### GIFアニメーション出力

損壊の様子や国旗の波打ちをGIFアニメーションとしてエンコードし、保存・共有できます。LZW圧縮によるGIF89a形式で、メディアンカット法による256色パレットを使用しています。

### 国際化

ブラウザの言語設定に応じて日本語・英語を自動切替します。

### モバイル対応

スマートフォンでもプレイ可能なレスポンシブレイアウトに対応しています。

## 隠し要素

- **旭日旗**: 0.1%の確率で旭日旗が生成され、軍艦行進曲が流れます
- **ニアミス検出**: 日本国旗に1色だけ異なる国旗が生成されると、壊れた君が代が流れます
- **サウンドテスト**: キーボードで `soundtest` と入力するとサウンドテストパネルが開きます

## 法的情報

本ゲームで表示される法的警告は以下の法令に基づいています。

- **刑法第92条（外国国章損壊等）**: 外国に対して侮辱を加える目的で国旗を損壊した場合、二年以下の拘禁刑又は二十万円以下の罰金
- **日本国国章損壊罪法案（第219回国会）**: 参政党が提出した日本国旗損壊を処罰する刑法改正案

本ゲームで生成される旗章は架空のものであり、既存の国旗との一致は偶然によるものです。

## 開発

```bash
docker compose up
# http://localhost:8088 でアクセス
```

ビルドステップは不要です。ES Modules を使用したバニラ JavaScript で構成されています。

## プロジェクト構成

```
├── index.html               # メインHTML・CSS
├── docker-compose.yml        # ローカル開発用（nginx:8088）
├── og-image.png / .svg       # OGP画像
└── src/
    ├── main.js               # ゲームロジック・状態管理
    ├── flag-renderer.js      # WebGL 国旗描画
    ├── shapes.js             # SDF 図形関数
    ├── i18n.js               # 国際化（日本語・英語）
    ├── sound.js              # 爆発音・燃焼音
    ├── kimigayo.js           # 君が代
    ├── gunkan.js             # 軍艦行進曲
    ├── anthems.js            # 各国国歌
    ├── soundtest.js          # サウンドテスト
    ├── flag-comparator.js    # 実在国旗マッチング
    ├── webgl-utils.js        # WebGL ユーティリティ
    ├── gif-encoder.js        # GIF89a エンコーダ
    └── effects/
        ├── base.js           # パーティクルエフェクト基底クラス
        ├── explosion.js      # 爆発エフェクト
        └── burn.js           # 燃焼エフェクト
```
